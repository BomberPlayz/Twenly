Buttons

Use Twenly's custom button styles in forms, actions, anywhere!

Simple buttons

<button class="ui button primary">primary button</button>
<button class="ui button secondary">secondary button</button>
<button class="ui button warning">warning button</button>
<button class="ui button danger">danger button</button>

You can also make anything into a button.

It can be useful when you want to make a link, or div look like a button.
