Horizontal menus

With Twenly's horizontal menus, you can easily create navbars, simple menus, etc!

To create a horizontal menu, you simply have to add the classes "ui menu" to a div.

Attention! Menus (both horizontal and vertical) use the "item" class in them, and not the "content" class!
Example
Log-in About us 

<div class="content">
    <div class="ui menu">
        <div class="item left">
            <button class="ui button primary">My projects</button>
        </div>
        <div class="item left">
            <button class="ui button primary">New project</button>
        </div>

        <a href="#" class="item">
            Log-in
        </a>
        <a href="#" class="item active">
            About us
        </a>

        <div class="item right">
            <input type="text" placeholder="search..."></input>
        </div>
    </div>
</div>

You can also put horizontal menus into itself.

With this, you can make cool, rounded sections of a horzontal menu, or have multiple active items, without the possibility of causing bugs.

Example
Log-in About us
Search projects Search assets 

<div class="ui menu">
    <div class="item left">
        <button class="ui button primary">My projects</button>
    </div>
    <div class="item left">
        <button class="ui button primary">New project</button>
    </div>

    <a href="#" class="item hover box">
        Log-in
    </a>
    <a href="#" class="item active">
        About us
    </a>

    <div class="ui menu right rounded">
        <a href="#" class="item active">
            Search projects
        </a>
        <a href="#" class="item">
            Search assets
        </a>
        <div class="item right">
            <button class="ui button primary">Log-out</button>
        </div>
        <div class="item right">
            <input type="text" placeholder="search..."></input>
        </div>
    </div>
</div>
